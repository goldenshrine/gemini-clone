# Gemini Clone (React + Vite)

This is a UI clone of Google's Gemini AI interface built using **React**, **Vite**, and **Tailwind CSS**. It's a minimal setup with Hot Module Reloading (HMR) and basic ESLint configuration.

## Tech Stack

- ⚛️ React
- ⚡ Vite
- 💅 Tailwind CSS
- ✅ ESLint

## Getting Started

```bash
npm install
npm run dev
```
